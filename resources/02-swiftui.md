# SwiftUI Resources (02)

Resources for rapid prototyping, state and data flow, navigation, animation, and building production-quality SwiftUI apps.

## Official Starting Points

- **[SwiftUI Tutorials (Apple)](https://developer.apple.com/tutorials/swiftui)** — Type: `Doc` | Level: `B`
  - Apple's official step-by-step tutorial series — best first stop
- **[SwiftUI Get Started](https://developer.apple.com/swiftui/get-started/)** — Type: `Doc` | Level: `B`
  - Quick overview and roadmap from Apple
